# KantoNPCs Documentation

KantoNPCs is an advanced NPC toolkit for Cobblemon adventure maps, multiplayer servers, and CobbleKanto-style projects.

This repository is used as the public documentation/manual for the mod. The main Modrinth/CurseForge page gives a general overview, while this documentation explains how the configurable NPC systems work.

## Recommended setup

1. Install KantoNPCs on both server and client.
2. Start the game/server once.
3. Let the mod generate its external files under `config/kantonpcs/`.
4. Configure NPCs in-game using the editor screens.
5. Use the documentation pages above when you need exact JSON formats, field behavior, or examples.

## What this documentation covers

This manual explains the main configurable systems included in KantoNPCs, including:

- Dialog NPCs
- Battle NPCs
- Rival NPCs
- Mart NPCs
- Trader NPCs
- Move Tutor NPCs
- NPC presets
- External skins
- Dialogue arrays
- Choice files
- Rewards
- Commands
- Money system
- Reset workflows
- Common setup problems

The main mod page is meant to be a general overview. This repository is meant to be the detailed setup reference for map makers, server owners, and modpack creators.

## Support

For bugs, compatibility issues, or unclear behavior, use the official issue tracker for the project:

https://github.com/PrimordioCobble/KantoNPCs-Wiki/issues

Please include:

- Minecraft version
- Fabric Loader version
- Fabric API version
- Cobblemon version
- KantoNPCs version
- Server/client logs if relevant
- Screenshots or short videos when the issue is visual
- Steps to reproduce the problem
